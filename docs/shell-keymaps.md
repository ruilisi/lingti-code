# Shell (Zsh) Keymaps

## Command Line Editing

### Navigation & History
| Keymap | Action | Mode |
|--------|--------|------|
| `Ctrl-R` | Vim mode and bash style historical reverse search | Zsh |
| `ESC C-x C-e` | Edit current command line in vim | Zsh |

### Alias Management
| Keymap | Action | Mode |
|--------|--------|------|
| `ae` | Edit aliases | Zsh |
| `ar` | Reload aliases | Zsh |
